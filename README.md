# 📻 Rust Radio TUI

Консольный радиоплеер с динамическим поиском станций. (Terminal radio player with dynamic station search.)

## ✨ Фичи
- 🔍 Поиск по [radio-browser.info](https://radio-browser.info) (жанр, страна, название)
- 🎨 Красивый TUI-интерфейс на `ratatui`
- 🔊 Управление громкостью через `wpctl` (PipeWire)
- ⚡ Мгновенный отклик, без задержек интерфейса
- 📦 Автономный бинарник (работает на любом Linux)
